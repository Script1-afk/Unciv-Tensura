# Unciv Tensura Mod v1

Bienvenue dans le mod **Tensura** pour Unciv, inspiré de l'univers de *Moi, quand je me réincarne en Slime*. Ce mod apporte une expérience complète et immersive en français.

## Contenu de la Version 1
- **Civilisations Majeures** :
  - **Fédération de Jura Tempest** (Rimuru) : Bonus de combat pour les monstres et bonus de culture.
  - **Royaume des Nains de Dwargon** (Gazel Dwargo) : Maîtres de la forge et de la production.
  - **Sainte République de Lubelius** (Luminous Valentine) : Puissance religieuse et bonus contre les monstres.

- **Unités Uniques** :
  - Loups-Tempête, Kijins, Chevaliers Sacrés, Chevaliers Nains.
  - **Veldora** : Unité héroïque dévastatrice.

- **Nouvelles Ressources** :
  - **Magicules** : Pour la science et la foi.
  - **Acier Magique** : Pour la production et l'équipement.
  - **Cœur de Dragon** : Ressource rare pour invoquer Veldora.

- **Arbre Technologique Étendu** :
  - Magie Élémentaire, Alchimie de Jura, Sagesse du Grand Sage, Invoquer un Seigneur Démon.

## Installation
1. Copiez l'URL de ce dépôt GitHub.
2. Dans Unciv, allez dans `Mods` > `Download mod from URL`.
3. Collez l'URL et cliquez sur `Download`.
4. Activez le mod dans les paramètres de la partie.

**Note** : Pour une expérience optimale, utilisez le mod de règles **Unciv-Tensura-Rules**.
